Exemplary Queries for CCF.OWL v2.0.1

We provide standard queries to a static version of the CCF API which includes CCF.OWL v2.0.1 and experimental data by five consortia made available as LOD. Note that counts for experimental data tissue blocks will be lower as in the paper because the database was compiled on June 23, 2022 and the paper reports data available on August 25, 2022.

Video Tutorials: Registration User Interface (RUI) and Exploration User Interface (EUI)

The RUI and EUI allow the 3D registration and subsequent exploration of tissue blocks in relation to the HRAv1.2 reference organs. Video Tutorial 1 and a standard operating procedure ("Using the CCF Registration User Interface") illustrate their usage. Video Tutorial 2 shows how to validate a RUI location with the EUI.

Video Demo: Organ VR Gallery

The CCF Organ VR Gallery is an immersive application that allows users to view, explore, and analyze 3D reference organs, anatomical structures, and cell types, in an integrated virtual reality (VR) environment. More information about this project can be found be found on the CCF Organ VR Gallery on the CCF Portal. A video demo is available below.
